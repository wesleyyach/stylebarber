# Barbearia Style

Landing page premium para uma barbearia de alto padrao, criada para transmitir sofistificacao, autoridade visual e foco em conversao para agendamentos.

[![JavaScript](https://img.shields.io/badge/JavaScript-111111?style=for-the-badge&logo=javascript&logoColor=F7DF1E)](#)

## Visao geral

O projeto apresenta uma experiencia digital luxuosa para barbearia, com secoes estrategicas de impacto visual, servicos, equipe, tabela de precos, galeria, depoimentos, mapa e contato direto via WhatsApp.

## Destaques

- Hero premium com foto real, CTA de agendamento e metricas animadas
- Menu responsivo com estado ativo por secao
- Cards de servicos com imagens reais e precificacao
- Secao de equipe com apresentacao dos barbeiros
- Tabela de planos com pacote em destaque
- Galeria fotografica com overlay cinematografico
- Depoimentos para reforco de prova social
- Formulario de agendamento com feedback visual
- Mapa integrado e botao flutuante de WhatsApp
- Layout responsivo para desktop, tablet e mobile

## Tecnologias

- HTML5 semantico
- CSS3 moderno com grid, efeitos glassmorphism, gradientes e responsividade
- JavaScript vanilla com Intersection Observer, contador animado, menu mobile e controle de formulario

## Como executar

1. Clone o repositorio:

```bash
git clone https://github.com/wesleyyach/stylebarber.git
```

2. Acesse a pasta do projeto:

```bash
cd stylebarber
```

3. Abra o arquivo `index.html` no navegador ou use a extensao Live Server no VS Code.

## Personalizacao rapida

- Troque o numero do WhatsApp em `index.html`
- Atualize textos, planos, nomes da equipe e endereco do mapa
- Substitua as imagens do Unsplash por fotos proprias da barbearia
- Ajuste as cores principais nas variaveis CSS dentro de `css/style.css`
